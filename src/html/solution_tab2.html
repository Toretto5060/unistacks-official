Solution

针对具体客户应用场景，提供定制化解决方案

  • 一体化监控平台
传统的IT应用监控基本上都是功能单一、分散的、无规范且繁琐的监控，并未能够给运维人员的监控运维工作带来质的提升。 一体化监控平台产品提供从数据的采集、交换、清洗、聚合分析到数据呈现一站式的大数据解决方案，支持文本文件、 数据源、console等多种采集方式，提供大数据的实时流数据分析能力，监控维度覆盖业务、应用服务、 中间件、主机等，能够从多种视角查看实时的监控数据，为运维工作带来极大的便利。

方案说明

业务实时监控

多维度指标一体化呈现，实时跟踪业务的数量、同比环比量、区域与渠道分布、响应时间及成功率等

数据库服务监控

支持关系型与非关系型数据库的监控，包括连接信息（连接时间、连接数）、SQL执行信息（耗时、频度、失败数）等

应用服务实时监控

实时跟踪与展示应用集群的总体业务请求信息（总量、正常量、缓慢量、异常量）和响应时长，实时展示Top N 节点的负载状况及趋势

告警故障定位跟踪

根据监控策略开展业务、集群节点、应用及中间件等目标监测，实时告警、等级判别、收集故障关联信息，支持多种方式告警

中间件运行监控

监测中间件堆栈使用量、线程数及 GC 开销状况，通过算法来推测程序是否存在内存泄漏、线程假死等问题

故障关联分析

针对告警和故障进行关联性识别，智能去重，减免干扰，为故障定位提供多维度日志信息及问题根因推断

主机存储实时监控

实时监测集群主机的总体性能信息，包括CPU、内存、存储、流量等视角的使用量、使用率和剩余量指标，实时展示 Top N 节点的性能趋势图

监控配置管理

满足运维的各类角色人员以不同视角来配置监控策略，支持配置目标对象、监测方法和频率、告警方式、接收人员、展示方式等

方案优势

一站式监控与操作

节省监控切换时间，监控注意力更加集中

基于业务运维

关注业务层面运维，支持更高层次的运维视角

聚焦故障

以预防故障、故障定位、处理故障为中心目标，实时监测、告警与故障跟踪

多维视角

满足不同角色监控人员的监控与操作需求

实时预测

支持业务、主机、应用等对象各类性能指标实时跟进展示及趋势预测

运维知识库

存储与整合运维经验库，智能提供运维操作的参考方案
